✈️ Flight Data Analysis Dashboard (SQL + Power BI)
📌 Overview

This project focuses on analyzing airline flight data using SQL queries and visualizing insights through an interactive Power BI dashboard.

It demonstrates:

Data extraction using SQL
Data transformation using Power Query (M Language)
Data visualization using Power BI
📂 Dataset

Table Name: airline_routes_100_with_profit

Key Columns:
RouteCode
Revenue
Cost
Profit
FlightDate 
OccupancyRate
FlightDuration
🛠️ Tech Stack
SQL
Power BI
Power Query (M Language)

🔄 Data Transformation (Power Query)
Data cleaning (removed null values)
Data type conversion
Created calculated columns
Applied filters
📊 Power BI Dashboard
🔹 Dashboard Features
Avg Occupancy by Aircraft (Donut Chart)
Total Profit by Month (Line Chart)
Total Cost vs Revenue (Bar Chart)
Profit by Route Code (Bar Chart)
Profit by Origin & Destination (Map)
Filters: Aircraft Type, Route Code, Month
